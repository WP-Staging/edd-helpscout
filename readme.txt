== Description ==

Easy Digital Downloads integration for HelpScout. Show purchase information right from your HelpScout.


== Installation ==

= Installing the plugin =
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for *HelpScout Easy Digital Downloads Integration* and click "Install now"
1. Alternatively, download the plugin and upload the contents of `helpscout-edd.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin
1. Set your HelpScout secret in the plugin settings.


Need help? Please take a look at the [frequently asked questions](http://wordpress.org/plugins/mailchimp-for-wp/faq/) first
